# Markowitz Portfolio Optimization Project

## Objective

This project evaluates a diversified six-stock portfolio using historical returns and Markowitz mean-variance portfolio methods. It estimates risk and return relationships in a training sample, compares simulated and optimization-based portfolios, traces the efficient frontier, and evaluates selected portfolios out of sample.

## Selected Stocks

The analysis uses the following stocks:

- AAPL
- JPM
- KO
- JNJ
- XOM
- BA

## Data and Train/Test Split

The project uses daily Yahoo Finance adjusted close price data. Log returns are computed from the adjusted close series.

The final available calendar month in the downloaded dataset is reserved as the out-of-sample test month. All earlier observations are used as the training sample for estimation, simulation, and optimization.

## Analyses Performed

The scripts perform the following analyses:

- Descriptive statistics for stock returns
- Correlation analysis with Pearson p-values and significance labels
- Markowitz simulation of random portfolio weights
- Optimization-based portfolio comparison, including minimum-variance and maximum-Sharpe portfolios
- Efficient frontier construction and visualization
- Out-of-sample evaluation using the held-out final calendar month

## Script Execution Order

Run the scripts from the project root in this order:

```bash
python src/01_download_data.py
python src/02_descriptive_analysis.py
python src/03_correlations.py
python src/04_markowitz_simulation.py
```

## Main Output Folders

- `data/` — raw and processed adjusted close prices and log returns
- `outputs/` — generated tables, portfolio weights, comparisons, and evaluation outputs
- `figures/` — generated plots and visualizations
- `report/` — project report materials
